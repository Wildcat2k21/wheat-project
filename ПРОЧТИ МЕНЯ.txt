(Если не читали редмишник, там подробнее)

Поднимите сервер MySQL, выполните sql скрипты в папке sql init files согласно порядку в MySQL Workbench:
1. database.sql
2. nodeuser.sql


установите зависимости:
npm install

запустите (Открыв терминал в корне): 
node index.js / nodemon index.js

В public находятся статичные ресурса сайта.
Основной скрипт сервера - index.js в корне.
Основные модули в папке modules

Все скрипты сопровождены комментариями.
